# hortinha

**README — Hortinha**

Criado por: Vitória de Oliveira Reis

Hortinha – Aplicativo de Gestão Colaborativa de Hortas em Condomínios  
Conexão com o ODS 2 – Fome Zero

O desenvolvimento mobile surge como resposta às necessidades atuais da sociedade por soluções práticas, escaláveis e inovadoras. Nesta atividade, eu idealizei e criei o Hortinha, um aplicativo conectado ao Objetivo de Desenvolvimento Sustentável (ODS) nº 2 da ONU, relacionado à Fome Zero e Agricultura Sustentável.

O objetivo do Hortinha é facilitar a autogestão de hortas colaborativas em condomínios, reduzindo o desperdício de alimentos causado pela falta de comunicação eficiente sobre quais alimentos foram colhidos, quais foram retirados para consumo e quais moradores participaram de cada ação.

O sistema incentiva transparência, organização comunitária e sustentabilidade.

**Objetivo Geral**

Oferecer um aplicativo funcional que permita gerenciar, de forma simples e colaborativa, os alimentos produzidos em hortas comunitárias.

**Objetivos Específicos**

Registrar itens colhidos e itens retirados para consumo.

Exibir produtos organizados por tipo e quantidade.

Ajustar quantidades com botões \+ e –.

Remover automaticamente produtos que chegam a zero.

Registrar histórico detalhado dos moradores.

Facilitar a navegação através de abas simples e intuitivas.

**Público-Alvo**

Moradores de condomínios com hortas comunitárias.

Comunidades urbanas de agricultura sustentável.

Escolas com hortas pedagógicas.

Projetos de redução de desperdício de alimentos.

**Tecnologias Utilizadas**

React

JavaScript (ES6+)

TailwindCSS

npm

**Estrutura do Aplicativo**  
Estante

Lista dos produtos disponíveis, organizados por categorias, com informação da colheita e controles de quantidade.

Adicionar

Formulário onde moradores podem registrar novos produtos colhidos, incluindo tipo, quantidade e data.

Moradores

Lista todos os moradores com suas atividades na horta.  
Mostra quem colheu, quem retirou, quanto e quando.

**Tipo de Aplicação**

Aplicação Web responsiva com possibilidade de PWA  
 Tecnologias: React \+ TailwindCSS

Justificativa:  
O código utiliza APIs e sintaxe do React para **web**, não usando bibliotecas de mobile nativo (React Native). Por isso, ele funciona perfeitamente em qualquer navegador moderno em Android e iOS, mas não é compilado como aplicativo mobile nativo.

**Requisitos do Sistema**

Cliente (Usuário final)

Android

Android 8.0 ou superior  
Navegador: Chrome, Edge, Firefox ou navegador padrão

iOS

iOS 12 ou superior  
Navegador: Safari (recomendado) ou Chrome

Pode ser instalado como PWA em ambos (Android e iOS 16+).

Para Desenvolvedores

Node.js 16+  
npm 7+  
Sistema operacional: Windows 10+, macOS 11+, Linux  
Navegador moderno para testes

**Conclusão**

O Hortinha promove organização, colaboração e redução do desperdício de alimentos, fortalecendo práticas comunitárias sustentáveis alinhadas ao ODS 2\.
